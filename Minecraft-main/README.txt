## Dependencias

Para ejecutar el servidor, instala las siguientes dependencias:

pip install moderngl
pip install pygame
pip install numba
pip install numpy
pip install PyGLM
pip install opensimplex

## Posibles errores

Si sale un error con las dependencias, abrir un cmd como administrador y hacer un pip upgrade
